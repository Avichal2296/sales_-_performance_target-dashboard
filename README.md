# Sales & Target Performance Dashboard

An interactive **Power BI dashboard** designed to monitor sales performance against predefined targets, enabling users to gain actionable insights through dynamic visualizations and data-driven analysis.

---

## Project Overview

The **Sales & Target Performance Dashboard** provides a comprehensive view of key sales metrics, helping users evaluate overall business performance, identify trends, and compare actual sales with targets across different dimensions.

The dashboard is built using **Microsoft Power BI** and leverages **Power Query** for data transformation and **DAX (Data Analysis Expressions)** for creating calculated measures and KPIs. Interactive features allow users to explore the data efficiently and make informed business decisions.

---

## Key Features

- Sales vs. Target Comparison
- Interactive KPI Cards
- Target Achievement Tracking
- Time-Based Sales Analysis
- Dynamic Filtering with Slicers
- Drill-Down Capabilities
- DAX Measures and Calculated Columns
- Data Cleaning & Transformation using Power Query
- Clean, Interactive, and User-Friendly Dashboard

1. Download or clone this repository.

4. Explore the dashboard using the interactive filters and visuals.
